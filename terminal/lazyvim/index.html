    LazyVim¶

    LazyVim は lazy.nvim をベースにした Neovim のフル機能スターターキット。 LSP・補完・ファジーファインダー・Git 連携などを標準搭載し、最小限の設定でモダンな開発環境を構築できる。

    インストール・アップデート・アンインストール¶

    事前要件の確認¶

    必要なツールを確認する

    # Neovim バージョン確認（0.9.0 以上が必要）
nvim --version

# 必須ツール
git --version
rg --version      # ripgrep（grep 検索用）
fd --version      # fd-find（ファイル検索用）

# 任意ツール
lazygit --version  # Git TUI（推奨）
node --version     # Node.js（一部 LSP で必要）

    LazyVim のインストール¶

    スターターテンプレートをクローンしてインストールする

    # 既存の Neovim 設定をバックアップ（念のため）
mv ~/.config/nvim ~/.config/nvim.bak
mv ~/.local/share/nvim ~/.local/share/nvim.bak

# LazyVim スターターをクローン
git clone https://github.com/LazyVim/starter ~/.config/nvim

# .git ディレクトリを削除（自分のリポジトリとして管理するため）
rm -rf ~/.config/nvim/.git

# Neovim を起動（初回は自動でプラグインをインストール）
nvim

    プラグインのアップデート¶

    :Lazy コマンドでプラグインを管理する

    " プラグインマネージャーを開く
:Lazy

" 全プラグインをアップデート
:Lazy update

" インストール・クリーン・アップデートを一括実行
:Lazy sync

" 未インストールのプラグインをインストール
:Lazy install

" 使用されていないプラグインを削除
:Lazy clean

" LazyVim の追加エクストラを管理する
:LazyExtras

    ヘルスチェック¶

    環境の問題を診断する

    " 総合ヘルスチェック
:checkhealth

" lazy.nvim のヘルスチェック
:checkhealth lazy

" LSP のヘルスチェック
:checkhealth lsp

    アンインストール¶

    LazyVim と関連データをすべて削除する

    # 設定ディレクトリを削除
rm -rf ~/.config/nvim

# データ・状態・キャッシュを削除
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim

    基本コマンド（モード操作）¶

    ノーマルモードへ戻る¶

    どのモードからでもノーマルモードへ戻る

    <Esc>       " ノーマルモードへ戻る（標準）
jk          " ノーマルモードへ戻る（LazyVim デフォルトのショートカット）

    インサートモード¶

    テキストを入力するモードに入る

    i           " カーソル位置の前から挿入
a           " カーソル位置の後から挿入
I           " 行頭（最初の非空白文字）から挿入
A           " 行末から挿入
o           " 現在行の下に新しい行を追加して挿入
O           " 現在行の上に新しい行を追加して挿入
s           " カーソル下の文字を削除して挿入
S           " 行全体を削除して挿入

    ビジュアルモード¶

    テキストを選択するモードに入る

    v           " ビジュアルモード（文字単位の選択）
V           " ビジュアルラインモード（行単位の選択）
<C-v>       " ビジュアルブロックモード（矩形選択）
gv          " 前回の選択範囲を再選択

    コマンドモード¶

    : でコマンドを入力する

    :           " コマンドラインモードへ
q:          " コマンド履歴を編集ウィンドウで開く

    カーソル移動¶

    基本移動¶

    ホームポジションのまま上下左右に移動する

    h           " 左へ移動
j           " 下へ移動
k           " 上へ移動
l           " 右へ移動

5j          " 5行下へ（数値＋移動コマンドで繰り返し）
10k         " 10行上へ

    単語単位の移動¶

    単語をまたいで素早く移動する

    w           " 次の単語の先頭へ
W           " 次の単語（空白区切り）の先頭へ
b           " 前の単語の先頭へ
B           " 前の単語（空白区切り）の先頭へ
e           " 次の単語の末尾へ
E           " 次の単語（空白区切り）の末尾へ
ge          " 前の単語の末尾へ

    行内移動¶

    現在の行の中で移動する

    0           " 行頭（列0）へ
^           " 行の最初の非空白文字へ
$           " 行末へ
g_          " 行末の最後の非空白文字へ
%           " 対応するカッコ・タグへジャンプ

    文字検索移動¶

    行内の文字を指定して移動する

    f{文字}     " 行内で次の {文字} へジャンプ（文字の上に移動）
F{文字}     " 行内で前の {文字} へジャンプ
t{文字}     " 行内で次の {文字} の直前へジャンプ
T{文字}     " 行内で前の {文字} の直後へジャンプ
;           " 直前の f/F/t/T を同方向で繰り返す
,           " 直前の f/F/t/T を逆方向で繰り返す

    ファイル全体の移動¶

    ファイルの先頭・末尾・任意の行へジャンプする

    gg          " ファイルの先頭へ
G           " ファイルの末尾へ
:50         " 50行目へジャンプ
50G         " 50行目へジャンプ（別表記）
H           " 画面の一番上の行へ
M           " 画面の中央の行へ
L           " 画面の一番下の行へ

    スクロール¶

    画面をスクロールしながら移動する

    <C-d>       " 半ページ下へスクロール
<C-u>       " 半ページ上へスクロール
<C-f>       " 1ページ下へスクロール
<C-b>       " 1ページ上へスクロール
zz          " カーソル行を画面中央に表示
zt          " カーソル行を画面上部に表示
zb          " カーソル行を画面下部に表示

    パラグラフ・セクション移動¶

    空行で区切られたブロック単位で移動する

    {           " 前のパラグラフ（空行）へ
}           " 次のパラグラフ（空行）へ
[[          " 前の関数・セクションの先頭へ
]]          " 次の関数・セクションの先頭へ

    ジャンプリスト¶

    過去にジャンプした場所を行き来する

    <C-o>       " 前のジャンプ位置へ戻る
<C-i>       " 次のジャンプ位置へ進む
``          " 直前のジャンプ元へ戻る

    編集操作¶

    テキストの削除¶

    文字・単語・行を削除する（削除内容はレジスタに保存される）

    x           " カーソル下の文字を削除
X           " カーソル前の文字を削除
dd          " 行全体を削除
D           " カーソルから行末まで削除
dw          " カーソルから次の単語の先頭まで削除
de          " カーソルから単語末尾まで削除
d$          " カーソルから行末まで削除
dgg         " カーソルからファイル先頭まで削除
dG          " カーソルからファイル末尾まで削除
di"         " ダブルクォートの内側を削除
da"         " ダブルクォートを含めて削除
di(         " カッコの内側を削除

    テキストの変更¶

    削除後すぐにインサートモードへ入る（c = change）

    cc          " 行全体を変更（行を削除してインサートモードへ）
C           " カーソルから行末まで変更
cw          " 単語を変更
ci"         " ダブルクォートの内側を変更
ca"         " ダブルクォートを含めて変更
ci(         " カッコの内側を変更
ci{         " 波カッコの内側を変更
ci[         " 角カッコの内側を変更
cit         " HTMLタグの内側を変更
r{文字}     " カーソル下の1文字を置換（インサートモードに入らない）
R           " 上書きモードに入る
~           " カーソル下の文字の大文字・小文字を切り替える
g~~         " 行全体の大文字・小文字を切り替える
gUw         " 単語を大文字にする
guw         " 単語を小文字にする

    コピー（ヤンク）と貼り付け¶

    テキストをコピーして別の場所に貼り付ける

    yy          " 行全体をコピー（ヤンク）
Y           " 行全体をコピー（yy と同じ）
yw          " 単語をコピー
y$          " カーソルから行末までコピー
yi"         " ダブルクォートの内側をコピー
yip         " パラグラフ全体をコピー

p           " カーソルの後（下）に貼り付け
P           " カーソルの前（上）に貼り付け
"0p         " ヤンクレジスタ（最後にコピーしたもの）を貼り付け
"+p         " システムクリップボードから貼り付け
"+y         " システムクリップボードにコピー

    アンドゥ・リドゥ¶

    操作を元に戻す・やり直す

    u           " 直前の操作を元に戻す（アンドゥ）
U           " 行への変更をすべて元に戻す
<C-r>       " アンドゥを取り消す（リドゥ）

    インデント¶

    コードのインデントを調整する

    >>          " 行のインデントを増やす
<<          " 行のインデントを減らす
==          " 行のインデントを自動修正
gg=G        " ファイル全体のインデントを自動修正
>ip         " パラグラフのインデントを増やす

" ビジュアルモードで選択後
>           " 選択範囲のインデントを増やす
<           " 選択範囲のインデントを減らす

    その他の編集操作¶

    行の結合・繰り返しなど便利な操作

    J           " 現在行と次の行を結合する
gJ          " スペースなしで行を結合する
.           " 直前の変更を繰り返す（ドットコマンド）
<C-a>       " カーソル下の数値をインクリメント（+1）
<C-x>       " カーソル下の数値をデクリメント（-1）

    検索・置換¶

    検索¶

    ファイル内のテキストを検索する

    /パターン   " 前方検索
?パターン   " 後方検索
n           " 次の一致へ（同方向）
N           " 前の一致へ（逆方向）
*           " カーソル下の単語を前方検索
#           " カーソル下の単語を後方検索
g*          " 単語境界なしで前方検索
:noh        " 検索ハイライトを消す（<leader>ur でも可）

    置換（substitute）¶

    :s コマンドでテキストを置換する

    :s/旧/新/           " 現在行の最初の一致を置換
:s/旧/新/g          " 現在行のすべての一致を置換
:%s/旧/新/g         " ファイル全体のすべての一致を置換
:%s/旧/新/gi        " 大文字・小文字を区別せず全体置換
:%s/旧/新/gc        " 確認しながら全体置換（y/n/a/q で応答）
:5,10s/旧/新/g      " 5〜10行目の範囲で置換
:'<,'>s/旧/新/g     " ビジュアル選択範囲で置換

    ファイル・バッファ操作¶

    ファイルの保存・終了¶

    ファイルを保存して終了する

    :w          " 保存
:w ファイル名  " 別名で保存
:q          " 終了（変更がある場合は失敗）
:q!         " 変更を破棄して強制終了
:wq         " 保存して終了
:x          " 変更がある場合だけ保存して終了
ZZ          " :x と同じ（ノーマルモードから）
ZQ          " :q! と同じ（ノーマルモードから）

    バッファ管理¶

    開いているファイル（バッファ）を切り替える

    :ls         " バッファ一覧を表示
:bn         " 次のバッファへ
:bp         " 前のバッファへ
:b{番号}    " 指定番号のバッファへ
:bd         " 現在のバッファを削除（ファイルは保持）
:bd!        " 変更を破棄してバッファを削除

" LazyVim デフォルトキーマップ
<S-h>       " 前のバッファへ（Shift+h）
<S-l>       " 次のバッファへ（Shift+l）
<leader>bd  " バッファを削除
<leader>bD  " 他のバッファをすべて削除

    ウィンドウ分割¶

    画面を分割して複数ファイルを同時に表示する

    :split      " 水平分割（:sp でも可）
:vsplit     " 垂直分割（:vsp でも可）
<C-w>s      " 水平分割（ショートカット）
<C-w>v      " 垂直分割（ショートカット）
<C-w>h      " 左のウィンドウへ移動
<C-w>j      " 下のウィンドウへ移動
<C-w>k      " 上のウィンドウへ移動
<C-w>l      " 右のウィンドウへ移動
<C-w>w      " 次のウィンドウへ移動（ローテーション）
<C-w>q      " 現在のウィンドウを閉じる
<C-w>=      " 全ウィンドウのサイズを均等にする
<C-w>>      " ウィンドウ幅を増やす
<C-w><      " ウィンドウ幅を減らす
<C-w>+      " ウィンドウ高さを増やす
<C-w>-      " ウィンドウ高さを減らす

    タブ管理¶

    Neovim のタブページを操作する

    :tabnew     " 新しいタブを開く
:tabnew ファイル名  " 指定ファイルを新しいタブで開く
gt          " 次のタブへ
gT          " 前のタブへ
{番号}gt    " 指定番号のタブへ
:tabclose   " 現在のタブを閉じる
:tabonly    " 現在のタブ以外をすべて閉じる

    Leader キーコマンド（LazyVim 標準）¶

    <leader> キーはデフォルトで スペース キーに割り当てられています。

    ファイル検索（Telescope / fzf-lua）¶

    <leader>f でファイルやテキストを素早く検索する

    <leader>ff  " ファイルを検索して開く（Find Files）
<leader>fr  " 最近使ったファイルを開く（Recent Files）
<leader>fg  " ファイル内テキストをライブ検索（Live Grep）
<leader>fb  " 開いているバッファ一覧を検索
<leader>fh  " ヘルプタグを検索
<leader>fR  " ワークスペース全体で最近使ったファイルを検索
<leader>fn  " 新しいファイルを作成
<leader>/   " 現在のバッファ内をグレップ
<leader>sg  " ワークスペース全体をグレップ

    上級者向けコマンド¶

    マクロ¶

    キー操作を録画して繰り返し実行する

    q{レジスタ} " マクロの録画を開始（例: qa）
q           " 録画を停止
@{レジスタ} " マクロを再生（例: @a）
@@          " 直前に再生したマクロを再実行
10@a        " マクロを10回繰り返す

    マーク¶

    ファイル内の位置にブックマークを付けて素早く戻る

    m{文字}     " マークを設定（例: ma）
`{文字}     " マークの行・列へジャンプ（例: `a）
'{文字}     " マークの行頭へジャンプ
``          " 最後のジャンプ位置へ戻る
:marks      " マーク一覧を表示
dm{文字}    " マークを削除

    レジスタ¶

    複数の内容を個別に保存して使い分ける

    :reg        " レジスタの内容を一覧表示
"{レジスタ}y  " 指定レジスタにヤンク（例: "ay）
"{レジスタ}p  " 指定レジスタから貼り付け（例: "ap）
"+y         " システムクリップボードにコピー
"+p         " システムクリップボードから貼り付け
"0          " 最後にヤンクした内容（yコマンドで上書きされない）
"_d         " ブラックホールレジスタへ削除（クリップボードに影響しない）

    外部コマンドの実行¶

    設定ファイル¶

    ディレクトリ構成¶

    LazyVim の設定ファイルは ~/.config/nvim/ 以下に配置する

    ~/.config/nvim/
├── init.lua               # エントリーポイント（基本は変更不要）
└── lua/
    ├── config/
    │   ├── autocmds.lua   # 自動コマンドの設定
    │   ├── keymaps.lua    # カスタムキーマップの設定
    │   ├── lazy.lua       # lazy.nvim の初期化設定
    │   └── options.lua    # Neovim オプションの設定
    └── plugins/
        ├── example.lua    # プラグインのカスタマイズ例
        └── *.lua          # 追加プラグインの設定ファイル

    options.lua のカスタマイズ例¶

    ~/.config/nvim/lua/config/options.lua に追記する

    -- Neovim の基本オプションをカスタマイズする
local opt = vim.opt

opt.relativenumber = true      -- 相対行番号を有効化
opt.scrolloff = 8              -- スクロール時に上下8行のマージンを確保
opt.colorcolumn = "80"         -- 80列目にガイドラインを表示
opt.wrap = false               -- 行の折り返しを無効化
opt.clipboard = "unnamedplus"  -- システムクリップボードと共有

    keymaps.lua のカスタマイズ例¶

    ~/.config/nvim/lua/config/keymaps.lua に追記する

    -- カスタムキーマップを追加する
local map = vim.keymap.set

-- ノーマルモード: jk で Esc の代替（すでに LazyVim に含まれるが例として）
-- map("i", "jk", "<Esc>", { desc = "インサートモードを抜ける" })

-- ノーマルモード: Ctrl+s で保存
map({ "n", "i", "v" }, "<C-s>", "<cmd>w<cr><esc>", { desc = "ファイルを保存" })

-- ノーマルモード: 行の移動（Alt+j/k）
map("n", "<A-j>", "<cmd>m .+1<cr>==", { desc = "行を下に移動" })
map("n", "<A-k>", "<cmd>m .-2<cr>==", { desc = "行を上に移動" })

    プラグインの追加・カスタマイズ例¶

    ~/.config/nvim/lua/plugins/ 以下に Lua ファイルを作成してプラグインを追加する

    -- ~/.config/nvim/lua/plugins/colorscheme.lua
-- カラースキームをカスタマイズする例
return {
  -- LazyVim のデフォルトカラースキームをカスタマイズ
  {
    "folke/tokyonight.nvim",
    opts = {
      style = "moon",  -- moon / storm / night / day
    },
  },
  -- カラースキームを変更する場合
  {
    "LazyVim/LazyVim",
    opts = {
      colorscheme = "tokyonight-moon",
    },
  },
}

require("lazy").setup({
  spec = {
    { "LazyVim/LazyVim", import = "lazyvim.plugins" },

# LazyVim

LazyVim は [lazy.nvim](https://github.com/folke/lazy.nvim) をベースにした Neovim のフル機能スターターキット。
LSP・補完・ファジーファインダー・Git 連携などを標準搭載し、最小限の設定でモダンな開発環境を構築できる。

---

## インストール・アップデート・アンインストール

### 事前要件の確認

!!! note "必要なツールを確認する"

```bash
# Neovim バージョン確認（0.9.0 以上が必要）
nvim --version

# 必須ツール
git --version
rg --version      # ripgrep（grep 検索用）
fd --version      # fd-find（ファイル検索用）

# 任意ツール
lazygit --version  # Git TUI（推奨）
node --version     # Node.js（一部 LSP で必要）
```

### LazyVim のインストール

!!! note "スターターテンプレートをクローンしてインストールする"

```bash
# 既存の Neovim 設定をバックアップ（念のため）
mv ~/.config/nvim ~/.config/nvim.bak
mv ~/.local/share/nvim ~/.local/share/nvim.bak

# LazyVim スターターをクローン
git clone https://github.com/LazyVim/starter ~/.config/nvim

# .git ディレクトリを削除（自分のリポジトリとして管理するため）
rm -rf ~/.config/nvim/.git

# Neovim を起動（初回は自動でプラグインをインストール）
nvim
```

### プラグインのアップデート

!!! note "`:Lazy` コマンドでプラグインを管理する"

```vim
" プラグインマネージャーを開く
:Lazy

" 全プラグインをアップデート
:Lazy update

" インストール・クリーン・アップデートを一括実行
:Lazy sync

" 未インストールのプラグインをインストール
:Lazy install

" 使用されていないプラグインを削除
:Lazy clean

" LazyVim の追加エクストラを管理する
:LazyExtras
```

### ヘルスチェック

!!! note "環境の問題を診断する"

```vim
" 総合ヘルスチェック
:checkhealth

" lazy.nvim のヘルスチェック
:checkhealth lazy

" LSP のヘルスチェック
:checkhealth lsp
```

### アンインストール

!!! note "LazyVim と関連データをすべて削除する"

```bash
# 設定ディレクトリを削除
rm -rf ~/.config/nvim

# データ・状態・キャッシュを削除
rm -rf ~/.local/share/nvim
rm -rf ~/.local/state/nvim
rm -rf ~/.cache/nvim
```

## 基本コマンド（モード操作）

### ノーマルモードへ戻る

!!! note "どのモードからでもノーマルモードへ戻る"

```vim
<Esc>       " ノーマルモードへ戻る（標準）
jk          " ノーマルモードへ戻る（LazyVim デフォルトのショートカット）
```

### インサートモード

!!! note "テキストを入力するモードに入る"

```vim
i           " カーソル位置の前から挿入
a           " カーソル位置の後から挿入
I           " 行頭（最初の非空白文字）から挿入
A           " 行末から挿入
o           " 現在行の下に新しい行を追加して挿入
O           " 現在行の上に新しい行を追加して挿入
s           " カーソル下の文字を削除して挿入
S           " 行全体を削除して挿入
```

### ビジュアルモード

!!! note "テキストを選択するモードに入る"

```vim
v           " ビジュアルモード（文字単位の選択）
V           " ビジュアルラインモード（行単位の選択）
<C-v>       " ビジュアルブロックモード（矩形選択）
gv          " 前回の選択範囲を再選択
```

### コマンドモード

!!! note "`:` でコマンドを入力する"

```vim
:           " コマンドラインモードへ
q:          " コマンド履歴を編集ウィンドウで開く
```

## カーソル移動

### 基本移動

!!! note "ホームポジションのまま上下左右に移動する"

```vim
h           " 左へ移動
j           " 下へ移動
k           " 上へ移動
l           " 右へ移動

5j          " 5行下へ（数値＋移動コマンドで繰り返し）
10k         " 10行上へ
```

### 単語単位の移動

!!! note "単語をまたいで素早く移動する"

```vim
w           " 次の単語の先頭へ
W           " 次の単語（空白区切り）の先頭へ
b           " 前の単語の先頭へ
B           " 前の単語（空白区切り）の先頭へ
e           " 次の単語の末尾へ
E           " 次の単語（空白区切り）の末尾へ
ge          " 前の単語の末尾へ
```

### 行内移動

!!! note "現在の行の中で移動する"

```vim
0           " 行頭（列0）へ
^           " 行の最初の非空白文字へ
$           " 行末へ
g_          " 行末の最後の非空白文字へ
%           " 対応するカッコ・タグへジャンプ
```

### 文字検索移動

!!! note "行内の文字を指定して移動する"

```vim
f{文字}     " 行内で次の {文字} へジャンプ（文字の上に移動）
F{文字}     " 行内で前の {文字} へジャンプ
t{文字}     " 行内で次の {文字} の直前へジャンプ
T{文字}     " 行内で前の {文字} の直後へジャンプ
;           " 直前の f/F/t/T を同方向で繰り返す
,           " 直前の f/F/t/T を逆方向で繰り返す
```

### ファイル全体の移動

!!! note "ファイルの先頭・末尾・任意の行へジャンプする"

```vim
gg          " ファイルの先頭へ
G           " ファイルの末尾へ
:50         " 50行目へジャンプ
50G         " 50行目へジャンプ（別表記）
H           " 画面の一番上の行へ
M           " 画面の中央の行へ
L           " 画面の一番下の行へ
```

### スクロール

!!! note "画面をスクロールしながら移動する"

```vim
<C-d>       " 半ページ下へスクロール
<C-u>       " 半ページ上へスクロール
<C-f>       " 1ページ下へスクロール
<C-b>       " 1ページ上へスクロール
zz          " カーソル行を画面中央に表示
zt          " カーソル行を画面上部に表示
zb          " カーソル行を画面下部に表示
```

### パラグラフ・セクション移動

!!! note "空行で区切られたブロック単位で移動する"

```vim
{           " 前のパラグラフ（空行）へ
}           " 次のパラグラフ（空行）へ
[[          " 前の関数・セクションの先頭へ
]]          " 次の関数・セクションの先頭へ
```

### ジャンプリスト

!!! note "過去にジャンプした場所を行き来する"

```vim
<C-o>       " 前のジャンプ位置へ戻る
<C-i>       " 次のジャンプ位置へ進む
``          " 直前のジャンプ元へ戻る
```

## 編集操作

### テキストの削除

!!! note "文字・単語・行を削除する（削除内容はレジスタに保存される）"

```vim
x           " カーソル下の文字を削除
X           " カーソル前の文字を削除
dd          " 行全体を削除
D           " カーソルから行末まで削除
dw          " カーソルから次の単語の先頭まで削除
de          " カーソルから単語末尾まで削除
d$          " カーソルから行末まで削除
dgg         " カーソルからファイル先頭まで削除
dG          " カーソルからファイル末尾まで削除
di"         " ダブルクォートの内側を削除
da"         " ダブルクォートを含めて削除
di(         " カッコの内側を削除
```

### テキストの変更

!!! note "削除後すぐにインサートモードへ入る（c = change）"

```vim
cc          " 行全体を変更（行を削除してインサートモードへ）
C           " カーソルから行末まで変更
cw          " 単語を変更
ci"         " ダブルクォートの内側を変更
ca"         " ダブルクォートを含めて変更
ci(         " カッコの内側を変更
ci{         " 波カッコの内側を変更
ci[         " 角カッコの内側を変更
cit         " HTMLタグの内側を変更
r{文字}     " カーソル下の1文字を置換（インサートモードに入らない）
R           " 上書きモードに入る
~           " カーソル下の文字の大文字・小文字を切り替える
g~~         " 行全体の大文字・小文字を切り替える
gUw         " 単語を大文字にする
guw         " 単語を小文字にする
```

### コピー（ヤンク）と貼り付け

!!! note "テキストをコピーして別の場所に貼り付ける"

```vim
yy          " 行全体をコピー（ヤンク）
Y           " 行全体をコピー（yy と同じ）
yw          " 単語をコピー
y$          " カーソルから行末までコピー
yi"         " ダブルクォートの内側をコピー
yip         " パラグラフ全体をコピー

p           " カーソルの後（下）に貼り付け
P           " カーソルの前（上）に貼り付け
"0p         " ヤンクレジスタ（最後にコピーしたもの）を貼り付け
"+p         " システムクリップボードから貼り付け
"+y         " システムクリップボードにコピー
```

### アンドゥ・リドゥ

!!! note "操作を元に戻す・やり直す"

```vim
u           " 直前の操作を元に戻す（アンドゥ）
U           " 行への変更をすべて元に戻す
<C-r>       " アンドゥを取り消す（リドゥ）
```

### インデント

!!! note "コードのインデントを調整する"

```vim
>>          " 行のインデントを増やす
<<          " 行のインデントを減らす
==          " 行のインデントを自動修正
gg=G        " ファイル全体のインデントを自動修正
>ip         " パラグラフのインデントを増やす

" ビジュアルモードで選択後
>           " 選択範囲のインデントを増やす
<           " 選択範囲のインデントを減らす
```

### その他の編集操作

!!! note "行の結合・繰り返しなど便利な操作"

```vim
J           " 現在行と次の行を結合する
gJ          " スペースなしで行を結合する
.           " 直前の変更を繰り返す（ドットコマンド）
<C-a>       " カーソル下の数値をインクリメント（+1）
<C-x>       " カーソル下の数値をデクリメント（-1）
```

## 検索・置換

### 検索

!!! note "ファイル内のテキストを検索する"

```vim
/パターン   " 前方検索
?パターン   " 後方検索
n           " 次の一致へ（同方向）
N           " 前の一致へ（逆方向）
*           " カーソル下の単語を前方検索
#           " カーソル下の単語を後方検索
g*          " 単語境界なしで前方検索
:noh        " 検索ハイライトを消す（<leader>ur でも可）
```

### 置換（substitute）

!!! note "`:s` コマンドでテキストを置換する"

```vim
:s/旧/新/           " 現在行の最初の一致を置換
:s/旧/新/g          " 現在行のすべての一致を置換
:%s/旧/新/g         " ファイル全体のすべての一致を置換
:%s/旧/新/gi        " 大文字・小文字を区別せず全体置換
:%s/旧/新/gc        " 確認しながら全体置換（y/n/a/q で応答）
:5,10s/旧/新/g      " 5〜10行目の範囲で置換
:'<,'>s/旧/新/g     " ビジュアル選択範囲で置換
```

## ファイル・バッファ操作

### ファイルの保存・終了

!!! note "ファイルを保存して終了する"

```vim
:w          " 保存
:w ファイル名  " 別名で保存
:q          " 終了（変更がある場合は失敗）
:q!         " 変更を破棄して強制終了
:wq         " 保存して終了
:x          " 変更がある場合だけ保存して終了
ZZ          " :x と同じ（ノーマルモードから）
ZQ          " :q! と同じ（ノーマルモードから）
```

### バッファ管理

!!! note "開いているファイル（バッファ）を切り替える"

```vim
:ls         " バッファ一覧を表示
:bn         " 次のバッファへ
:bp         " 前のバッファへ
:b{番号}    " 指定番号のバッファへ
:bd         " 現在のバッファを削除（ファイルは保持）
:bd!        " 変更を破棄してバッファを削除

" LazyVim デフォルトキーマップ
<S-h>       " 前のバッファへ（Shift+h）
<S-l>       " 次のバッファへ（Shift+l）
<leader>bd  " バッファを削除
<leader>bD  " 他のバッファをすべて削除
```

### ウィンドウ分割

!!! note "画面を分割して複数ファイルを同時に表示する"

```vim
:split      " 水平分割（:sp でも可）
:vsplit     " 垂直分割（:vsp でも可）
<C-w>s      " 水平分割（ショートカット）
<C-w>v      " 垂直分割（ショートカット）
<C-w>h      " 左のウィンドウへ移動
<C-w>j      " 下のウィンドウへ移動
<C-w>k      " 上のウィンドウへ移動
<C-w>l      " 右のウィンドウへ移動
<C-w>w      " 次のウィンドウへ移動（ローテーション）
<C-w>q      " 現在のウィンドウを閉じる
<C-w>=      " 全ウィンドウのサイズを均等にする
<C-w>>      " ウィンドウ幅を増やす
<C-w><      " ウィンドウ幅を減らす
<C-w>+      " ウィンドウ高さを増やす
<C-w>-      " ウィンドウ高さを減らす
```

### タブ管理

!!! note "Neovim のタブページを操作する"

```vim
:tabnew     " 新しいタブを開く
:tabnew ファイル名  " 指定ファイルを新しいタブで開く
gt          " 次のタブへ
gT          " 前のタブへ
{番号}gt    " 指定番号のタブへ
:tabclose   " 現在のタブを閉じる
:tabonly    " 現在のタブ以外をすべて閉じる
```

<leader>fb  " 開いているバッファ一覧を検索
<leader>fh  " ヘルプタグを検索

<leader>ss  " ドキュメント内のシンボルを検索
<leader>sS  " ワークスペース内のシンボルを検索
<leader>sd  " 診断（エラー・警告）を検索

## Flash.nvim（高速ジャンプ）

### Flash によるジャンプ

!!! note "画面上の任意の位置へラベルで即座にジャンプする"

```vim
s           " Flash ジャンプを起動（文字を入力してラベル選択）
S           " Flash Treesitter ジャンプ（構文ノードを選択）

" オペレータ待機モード中
r           " リモート Flash（離れた場所の操作対象を指定）
R           " Treesitter 検索
```

## 上級者向けコマンド

### マクロ

!!! note "キー操作を録画して繰り返し実行する"

```vim
q{レジスタ} " マクロの録画を開始（例: qa）
q           " 録画を停止
@{レジスタ} " マクロを再生（例: @a）
@@          " 直前に再生したマクロを再実行
10@a        " マクロを10回繰り返す
```

### マーク

!!! note "ファイル内の位置にブックマークを付けて素早く戻る"

```vim
m{文字}     " マークを設定（例: ma）
`{文字}     " マークの行・列へジャンプ（例: `a）
'{文字}     " マークの行頭へジャンプ
``          " 最後のジャンプ位置へ戻る
:marks      " マーク一覧を表示
dm{文字}    " マークを削除
```

### レジスタ

!!! note "複数の内容を個別に保存して使い分ける"

```vim
:reg        " レジスタの内容を一覧表示
"{レジスタ}y  " 指定レジスタにヤンク（例: "ay）
"{レジスタ}p  " 指定レジスタから貼り付け（例: "ap）
"+y         " システムクリップボードにコピー
"+p         " システムクリップボードから貼り付け
"0          " 最後にヤンクした内容（yコマンドで上書きされない）
"_d         " ブラックホールレジスタへ削除（クリップボードに影響しない）
```

### 折りたたみ（Fold）

!!! note "コードブロックを折りたたんで表示を整理する"

```vim
zf{motion}  " 手動で折りたたみを作成（例: zf5j）
zo          " 折りたたみを開く
zc          " 折りたたみを閉じる
za          " 折りたたみをトグル
zR          " すべての折りたたみを開く
zM          " すべての折りたたみを閉じる
zr          " 折りたたみレベルを1段階開く
zm          " 折りたたみレベルを1段階閉じる
zd          " 折りたたみを削除
```

### Treesitter テキストオブジェクト

!!! note "コードの構造（関数・クラスなど）を単位として選択・操作する"

```vim
vaf         " 関数全体を選択（around function）
vif         " 関数の本体を選択（inner function）
vac         " クラス全体を選択（around class）
vic         " クラスの本体を選択（inner class）
vaa         " 引数全体を選択（around argument/parameter）
via         " 引数の内側を選択（inner argument）
```

### サラウンド操作（mini.surround）

!!! note "テキストを囲む記号を追加・変更・削除する"

```vim
ysiw"       " 単語をダブルクォートで囲む（ys + motion + 囲む文字）
ysiw(       " 単語を () で囲む
yss"        " 行全体をダブルクォートで囲む
cs"'        " ダブルクォートをシングルクォートに変更（change surround）
cs({        " () を {} に変更
ds"         " ダブルクォートを削除（delete surround）
ds(         " () を削除
```

### スペルチェック

!!! note "スペルチェック機能を使ってタイポを修正する"

```vim
:set spell      " スペルチェックを有効化
:set nospell    " スペルチェックを無効化
]s              " 次のスペルミスへ
[s              " 前のスペルミスへ
z=              " スペル修正候補を表示
zg              " 単語をスペル辞書に追加（good）
zw              " 単語を誤りとしてマーク（wrong）
```

## 知ってると便利なコマンド

### 外部コマンドの実行

!!! note "Neovim を終了せずにシェルコマンドを実行する"

```vim
:!{コマンド}     " シェルコマンドを実行（例: :!ls）
:r !{コマンド}   " コマンドの出力をカーソル位置に挿入（例: :r !date）
:.!{コマンド}    " 現在行をコマンドの入力として渡し、出力で置換
:%!python3 -m json.tool  " ファイル全体を Python の JSON フォーマッタに通す
```

### 差分（diff）表示

!!! note "2つのファイルや変更前後を比較する"

```vim
nvim -d ファイル1 ファイル2   " 2ファイルを diff モードで開く
:windo diffthis               " 分割している2ウィンドウを diff 比較
:diffoff                      " diff モードを終了
]c                            " 次の差分へ
[c                            " 前の差分へ
do                            " 他ウィンドウから現在ウィンドウへ変更を取得（diff obtain）
dp                            " 現在ウィンドウから他ウィンドウへ変更を適用（diff put）
```

### セッション管理（persistence.nvim）

!!! note "作業状態を保存して後で再開する"

```vim
<leader>qs  " セッションを保存
<leader>ql  " 最後のセッションを復元
<leader>qd  " セッションを削除
```

### ファイルタイプ・エンコーディング

!!! note "ファイルの種類やエンコーディングを設定する"

```vim
:set filetype=python    " ファイルタイプを Python に設定
:set filetype=markdown  " ファイルタイプを Markdown に設定
:set fileencoding=utf-8 " エンコーディングを UTF-8 に設定
:file                   " 現在のファイル名と状態を表示
```

### which-key（キーマップの確認）

!!! note "使えるキーマップを動的に確認する"

```vim
<leader>    " スペースキーを押すと which-key がポップアップ
<leader>?   " バッファローカルのキーマップを一覧表示
```

### 便利な移動・編集

!!! note "知っておくと日常の編集が速くなるコマンド"

```vim
gf          " カーソル下のファイルパスをファイルとして開く
gx          " カーソル下のURLをブラウザで開く
ga          " カーソル下の文字のASCIIコードを表示
g?          " ROT13エンコードをトグル
:sort       " 選択範囲を並び替え
:sort!      " 逆順で並び替え
:sort u     " 重複行を削除して並び替え
:g/パターン/d   " パターンに一致する行をすべて削除
:g/パターン/y A " パターンに一致する行をすべてレジスタaにコピー
:v/パターン/d   " パターンに一致しない行を削除（逆grep）
```

## オプション

### 表示設定

| オプション | 説明 |
| --- | --- |
| `:set number` | 行番号を表示 |
| `:set nonumber` | 行番号を非表示 |
| `:set relativenumber` | 相対行番号を表示 |
| `:set cursorline` | カーソル行をハイライト |
| `:set wrap` | 行を折り返して表示 |
| `:set nowrap` | 行を折り返さない |
| `:set list` | 不可視文字（タブ・改行など）を表示 |
| `:set signcolumn=yes` | サインカラム（エラーアイコン列）を常に表示 |
| `:colorscheme {名前}` | カラースキームを変更（例: `:colorscheme tokyonight`） |

### 検索設定

| オプション | 説明 |
| --- | --- |
| `:set ignorecase` | 検索で大文字・小文字を区別しない |
| `:set smartcase` | 検索パターンに大文字があれば区別する |
| `:set hlsearch` | 検索結果をハイライト |
| `:set incsearch` | インクリメンタル検索を有効化 |

### インデント・タブ設定

| オプション | 説明 |
| --- | --- |
| `:set tabstop=2` | タブ幅を2スペースに設定 |
| `:set shiftwidth=2` | インデント幅を2スペースに設定 |
| `:set expandtab` | タブを空白に展開 |
| `:set autoindent` | 自動インデントを有効化 |

## 設定ファイル

### ディレクトリ構成

!!! note "LazyVim の設定ファイルは `~/.config/nvim/` 以下に配置する"

```text
~/.config/nvim/
├── init.lua               # エントリーポイント（基本は変更不要）
└── lua/
    ├── config/
    │   ├── autocmds.lua   # 自動コマンドの設定
    │   ├── keymaps.lua    # カスタムキーマップの設定
    │   ├── lazy.lua       # lazy.nvim の初期化設定
    │   └── options.lua    # Neovim オプションの設定
    └── plugins/
        ├── example.lua    # プラグインのカスタマイズ例
        └── *.lua          # 追加プラグインの設定ファイル
```

### options.lua のカスタマイズ例

!!! note "~/.config/nvim/lua/config/options.lua に追記する"

```lua
-- Neovim の基本オプションをカスタマイズする
local opt = vim.opt

opt.relativenumber = true      -- 相対行番号を有効化
opt.scrolloff = 8              -- スクロール時に上下8行のマージンを確保
opt.colorcolumn = "80"         -- 80列目にガイドラインを表示
opt.wrap = false               -- 行の折り返しを無効化
opt.clipboard = "unnamedplus"  -- システムクリップボードと共有
```

### keymaps.lua のカスタマイズ例

!!! note "~/.config/nvim/lua/config/keymaps.lua に追記する"

```lua
-- カスタムキーマップを追加する
local map = vim.keymap.set

-- ノーマルモード: jk で Esc の代替（すでに LazyVim に含まれるが例として）
-- map("i", "jk", "<Esc>", { desc = "インサートモードを抜ける" })

-- ノーマルモード: Ctrl+s で保存
map({ "n", "i", "v" }, "<C-s>", "<cmd>w<cr><esc>", { desc = "ファイルを保存" })

-- ノーマルモード: 行の移動（Alt+j/k）
map("n", "<A-j>", "<cmd>m .+1<cr>==", { desc = "行を下に移動" })
map("n", "<A-k>", "<cmd>m .-2<cr>==", { desc = "行を上に移動" })
```

### プラグインの追加・カスタマイズ例

!!! note "~/.config/nvim/lua/plugins/ 以下に Lua ファイルを作成してプラグインを追加する"

```lua
-- ~/.config/nvim/lua/plugins/colorscheme.lua
-- カラースキームをカスタマイズする例
return {
  -- LazyVim のデフォルトカラースキームをカスタマイズ
  {
    "folke/tokyonight.nvim",
    opts = {
      style = "moon",  -- moon / storm / night / day
    },
  },
  -- カラースキームを変更する場合
  {
    "LazyVim/LazyVim",
    opts = {
      colorscheme = "tokyonight-moon",
    },
  },
}
```

### LazyExtras でエクストラを有効化

!!! note "`:LazyExtras` から GUI でエクストラを有効・無効にできる"

```vim
" LazyExtras を開く
:LazyExtras
```

```lua
-- ~/.config/nvim/lua/config/lazy.lua で直接指定することも可能
require("lazy").setup({
  spec = {
    { "LazyVim/LazyVim", import = "lazyvim.plugins" },
    -- エクストラを有効化する例
    { import = "lazyvim.plugins.extras.lang.python" },
    { import = "lazyvim.plugins.extras.lang.typescript" },
    { import = "lazyvim.plugins.extras.lang.json" },
    { import = "lazyvim.plugins.extras.ui.mini-animate" },
    { import = "plugins" },
  },
})
```

## 頻出コマンド早引き

```vim
" ===== モード操作 =====
i               " インサートモードへ
<Esc> / jk      " ノーマルモードへ戻る
v / V / <C-v>   " ビジュアル / ラインビジュアル / ブロックビジュアルモードへ

" ===== 保存・終了 =====
:w              " 保存
:q              " 終了
:wq / ZZ        " 保存して終了
:q!             " 強制終了

" ===== 移動 =====
gg / G          " ファイル先頭 / 末尾へ
<C-d> / <C-u>   " 半ページ下 / 上
<C-o> / <C-i>   " ジャンプ履歴を戻る / 進む
%               " 対応するカッコへジャンプ

" ===== 編集 =====
u / <C-r>       " アンドゥ / リドゥ
.               " 直前の操作を繰り返す
dd / yy / p     " 行を削除 / コピー / 貼り付け
ci" / ci( / ci{ " ダブルクォート / () / {} の内側を変更
>>  / <<        " インデントを増やす / 減らす
gg=G            " ファイル全体のインデントを自動修正

" ===== 検索・置換 =====
/パターン       " 前方検索
n / N           " 次 / 前の一致へ
*               " カーソル下の単語を検索
:%s/旧/新/g     " ファイル全体を置換

" ===== Leader キー（スペースキー） =====
<leader>ff      " ファイル検索
<leader>fg      " ライブ grep
<leader>e       " ファイルツリーをトグル
<leader>gg      " LazyGit を開く
<leader>ca      " コードアクション
<leader>cr      " シンボルをリネーム
gd / gr / K     " 定義へジャンプ / 参照 / ホバードキュメント

" ===== LSP 診断 =====
]d / [d         " 次 / 前の診断へ

" ===== バッファ =====
<S-h> / <S-l>   " 前 / 次のバッファへ
<leader>bd      " バッファを削除

" ===== プラグイン管理 =====
:Lazy           " プラグインマネージャーを開く
:Lazy update    " プラグインをアップデート
:LazyExtras     " エクストラを管理
:checkhealth    " ヘルスチェック
```

# Bash

日常的な作業から高度な処理まで、幅広く活用できるBashコマンドを目的別にまとめています。

## 基本コマンド

### ls

ディレクトリの中身を一覧表示します。

```bash
# カレントディレクトリのファイルとディレクトリを表示する
ls

# 隠しファイルを含めて詳細情報をリスト形式で表示する
ls -la
```

### cd

作業しているディレクトリを移動します。

```bash
# 指定した絶対パスのディレクトリに移動する
cd /var/log

# 1つ上の階層のディレクトリに移動する
cd ..

# ホームディレクトリに戻る
cd ~
```

### pwd

現在の作業ディレクトリ（カレントディレクトリ）の絶対パスを表示します。

```bash
# 現在のパスを確認する
pwd
```

### mkdir

新しいディレクトリを作成します。

```bash
# 新しいディレクトリを作成する
mkdir new_directory

# 深い階層のディレクトリを中間のディレクトリも含めて一度に作成する
mkdir -p parent/child/grandchild
```

### rm

ファイルやディレクトリを削除します。

```bash
# ファイルを削除する
rm file.txt

# ディレクトリとその中身を警告なしで強制的に削除する
rm -rf target_directory
```

## よく使うコマンド

### cp

ファイルやディレクトリをコピーします。

```bash
# ファイルをコピーする
cp original.txt backup.txt

# ディレクトリとその中身を再帰的にすべてコピーする
cp -r source_dir destination_dir
```

### mv

ファイルやディレクトリを移動、または名前を変更します。

```bash
# ファイルの名前を変更する
mv old_name.txt new_name.txt

# ファイルを別のディレクトリに移動する
mv file.txt /path/to/destination/
```

### cat

ファイルの内容をターミナル上に表示します。複数ファイルの結合にも使われます。

```bash
# ファイルの内容を表示する
cat config.yml

# 2つのファイルを結合して新しいファイルを作成する
cat file1.txt file2.txt > combined.txt
```

### grep

ファイルの中から特定の文字列（パターン）を検索し、該当する行を表示します。

```bash
# ファイルから特定の文字列を検索する
grep "error" server.log

# 大文字と小文字を区別せずに検索し、該当した行番号も表示する
grep -in "warning" app.log
```

### find

条件（ファイル名、サイズ、更新日時など）を指定してファイルやディレクトリを検索します。

```bash
# カレントディレクトリ以下から拡張子が.txtのファイルを検索する
find . -name "*.txt"

# サイズが100MB以上のファイルをルートディレクトリから検索する
find / -size +100M
```

## 上級者向けコマンド

### awk

空白や特定の文字で区切られたテキストデータを処理・抽出するのに非常に強力なコマンドです。

```bash
# スペース区切りのテキストデータから2列目だけを表示する
awk '{print $2}' data.txt

# カンマ区切りのCSVファイルから1列目と3列目を表示する
awk -F ',' '{print $1, $3}' users.csv
```

### sed

テキストストリームをフィルタリングしたり、正規表現を用いて文字列を置換したりするエディタです。

```bash
# ファイル内の「apple」を最初の1つだけ「orange」に置換して標準出力に表示する
sed 's/apple/orange/' fruits.txt

# ファイル内の「apple」をすべて「orange」に置換し、ファイルを直接上書き保存する
sed -i 's/apple/orange/g' fruits.txt
```

### xargs

標準入力からデータを受け取り、それを引数として別のコマンドを実行します。パイプ（|）と組み合わせて使います。

```bash
# findで見つけた.bakファイルをxargsに渡し、すべて削除する
find . -name "*.bak" | xargs rm

# テキストファイルに1行ずつ記載されたURLをすべてダウンロードする
cat urls.txt | xargs wget
```

### tar

複数のファイルやディレクトリを1つのアーカイブファイルにまとめたり、圧縮・展開したりします。

```bash
# ディレクトリをgzip形式で圧縮してアーカイブを作成する
tar -czvf backup.tar.gz target_directory/

# 圧縮されたアーカイブを展開する
tar -xzvf backup.tar.gz
```

### vim

ターミナル上で動作する強力なテキストエディタです。設定ファイルやスクリプトをターミナル内で直接編集・作成する際に不可欠です。

```bash
# ファイルをvimで開く（ファイルが存在しない場合は新規作成モードになる）
vim script.sh

# ファイルを開いた後、テキストを入力するには「i」キーを押して入力モード（INSERT）に切り替える
# 編集が終わったら「Esc」キーを押し、「:wq」と入力してEnterキーを押すことで保存して終了する
```

## 知ってると便利なコマンド

### history

これまでに実行したコマンドの履歴を一覧表示します。

```bash
# コマンド履歴を表示する
history

# 履歴の中から過去に実行した「ssh」を含むコマンドを検索する
history | grep "ssh"
```

### alias

長くて複雑なコマンドや、よく使うオプション付きのコマンドを短い別名で登録します。

```bash
# ls -laを「ll」という短いコマンドで実行できるように登録する
alias ll='ls -la'

# 現在登録されているエイリアスの一覧を表示する
alias
```

### df

システム上のディスクの空き容量やファイルシステムの使用状況を確認します。

```bash
# ディスクの使用状況を人間が読みやすい形式（GBやMB単位）で表示する
df -h
```

### tail

ファイルの末尾部分（デフォルトは10行）を表示します。ログファイルのリアルタイム監視によく使われます。

```bash
# ファイルの末尾20行を表示する
tail -n 20 error.log

# ファイルに新しく追記される内容をリアルタイムで監視し続ける
tail -f /var/log/syslog
```

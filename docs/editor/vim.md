# Vim

Vim の主要コマンドをまとめたリファレンスです。  
モードの概念を理解してから各コマンドを習得すると効果的です。

---

## Vimのモードについて

Vim には複数のモードがあります。

| モード               | 説明                         | 切り替え方         |
| -------------------- | ---------------------------- | ------------------ |
| ノーマルモード       | 移動・操作の基本モード       | `Esc`              |
| インサートモード     | 文字入力モード               | `i`, `a`, `o` など |
| ビジュアルモード     | テキスト選択モード           | `v`, `V`, `Ctrl+v` |
| コマンドラインモード | 保存・検索などのコマンド入力 | `:`                |

## 基本コマンド

### ファイルの開き方・終了

| コマンド          | 説明                            |
| ----------------- | ------------------------------- |
| `vim ファイル名`  | ファイルを開く                  |
| `:q`              | Vimを終了する（変更がない場合） |
| `:q!`             | 変更を破棄して強制終了する      |
| `:w`              | ファイルを上書き保存する        |
| `:wq` または `ZZ` | 保存して終了する                |
| `:w ファイル名`   | 別名で保存する                  |
| `:e ファイル名`   | 別のファイルを開く              |

**例：** 新規ファイルを作成して保存する

```bash
vim memo.md
```

Vimが開いたら `i` でインサートモードに入り、文字を入力。  
終わったら `Esc` → `:wq` で保存して終了。

### モードの切り替え

| コマンド | 説明                                             |
| -------- | ------------------------------------------------ |
| `i`      | カーソル位置の前からインサートモードへ           |
| `I`      | 行頭からインサートモードへ                       |
| `a`      | カーソル位置の後ろからインサートモードへ         |
| `A`      | 行末からインサートモードへ                       |
| `o`      | 現在行の下に新しい行を追加してインサートモードへ |
| `O`      | 現在行の上に新しい行を追加してインサートモードへ |
| `s`      | カーソル下の文字を削除してインサートモードへ     |
| `S`      | 現在行を削除してインサートモードへ               |
| `Esc`    | ノーマルモードに戻る                             |

**例：** 行末に追記する

```text
（ノーマルモードで）
A → 行末に移動してインサートモードに入る
「追記したいテキスト」と入力
Esc → ノーマルモードに戻る
```

| コマンド | 説明                                 |
| -------- | ------------------------------------ |
| `h`      | 左に移動                             |

| `l`      | 右に移動                             |
| `0`      | 行頭に移動                           |

| `gg`     | ファイルの先頭に移動                 |

| `b`      | 前の単語の先頭に移動                 |
| `e`      | 次の単語の末尾に移動                 |

**例：** 100行目に移動する

```text
100G
```

## よく使うコマンド

### テキストの削除

| コマンド        | 説明                                      |
| --------------- | ----------------------------------------- |
| `x`             | カーソル下の1文字を削除                   |
| `X`             | カーソルの前の1文字を削除                 |
| `dd`            | 現在行を丸ごと削除                        |
| `数字dd`        | 指定した行数を削除（例：`3dd` で3行削除） |
| `dw`            | カーソル位置から次の単語まで削除          |
| `d$` または `D` | カーソル位置から行末まで削除              |
| `d0`            | カーソル位置から行頭まで削除              |
| `dgg`           | カーソル位置からファイル先頭まで削除      |
| `dG`            | カーソル位置からファイル末尾まで削除      |

**例：** 現在行から5行まとめて削除する

```text
5dd
```

### テキストのコピー・貼り付け

| コマンド        | 説明                                          |
| --------------- | --------------------------------------------- |
| `yy` または `Y` | 現在行をコピー（ヤンク）                      |
| `数字yy`        | 指定した行数をコピー（例：`3yy` で3行コピー） |
| `yw`            | カーソル位置から次の単語までコピー            |
| `y$`            | カーソル位置から行末までコピー                |
| `p`             | カーソルの後ろ（下）に貼り付け                |
| `P`             | カーソルの前（上）に貼り付け                  |

**例：** 現在行を複製する

```text
yy → pp
（現在行をコピーして、その後ろに2回貼り付け）
```

### 元に戻す・やり直す

| コマンド   | 説明                             |
| ---------- | -------------------------------- |
| `u`        | 直前の操作を元に戻す（アンドゥ） |
| `Ctrl + r` | やり直す（リドゥ）               |
| `U`        | 現在行への変更をすべて元に戻す   |

**例：** 誤って行を削除した場合

```text
u → 削除を取り消す
```

### 検索と置換

| コマンド       | 説明                               |
| -------------- | ---------------------------------- |
| `/検索ワード`  | 下方向に検索                       |
| `?検索ワード`  | 上方向に検索                       |
| `n`            | 次の検索結果に移動                 |
| `N`            | 前の検索結果に移動                 |
| `:s/旧/新/`    | 現在行の最初のマッチを置換         |
| `:s/旧/新/g`   | 現在行のすべてのマッチを置換       |
| `:%s/旧/新/g`  | ファイル全体のすべてのマッチを置換 |
| `:%s/旧/新/gc` | ファイル全体を確認しながら置換     |

**例：** ファイル全体の `foo` を `bar` に置換する

```text
:%s/foo/bar/g
```

**例：** 一つずつ確認しながら置換する

```text
:%s/foo/bar/gc
（y: 置換する / n: スキップ / a: すべて置換 / q: 終了）
```

### 変更コマンド（チェンジ）

| コマンド        | 説明                                                           |
| --------------- | -------------------------------------------------------------- |
| `cw`            | カーソル位置から次の単語まで変更（削除してインサートモードへ） |
| `cc` または `S` | 現在行を変更                                                   |
| `c$` または `C` | カーソル位置から行末まで変更                                   |
| `r文字`         | カーソル下の1文字を指定した文字に置き換え                      |
| `R`             | 上書きモードに入る                                             |

**例：** 単語を素早く書き換える

```text
カーソルを単語の先頭に置いて cw → 新しい単語を入力 → Esc
```

## 上級者向けコマンド

### ビジュアルモード

| コマンド   | 説明                             |
| ---------- | -------------------------------- |
| `v`        | 文字単位のビジュアルモード       |
| `V`        | 行単位のビジュアルモード         |
| `Ctrl + v` | 矩形（ブロック）ビジュアルモード |
| `gv`       | 直前の選択範囲を再選択           |
| 選択後 `y` | 選択範囲をコピー                 |
| 選択後 `d` | 選択範囲を削除                   |
| 選択後 `c` | 選択範囲を変更                   |
| 選択後 `>` | 選択範囲をインデント（右）       |
| 選択後 `<` | 選択範囲をインデント解除（左）   |
| 選択後 `~` | 大文字・小文字を切り替え         |
| 選択後 `u` | 小文字に変換                     |
| 選択後 `U` | 大文字に変換                     |

**例：** 3行をまとめてインデントする

```text
V → jj → （3行選択） → >
```

---

### 矩形ビジュアルモードの活用

| コマンド                    | 説明                           |
| --------------------------- | ------------------------------ |
| `Ctrl + v` → 選択 → `I`     | 選択した各行の先頭に文字を挿入 |
| `Ctrl + v` → 選択 → `A`     | 選択した各行の末尾に文字を挿入 |
| `Ctrl + v` → 選択 → `d`     | 矩形範囲を削除                 |
| `Ctrl + v` → 選択 → `r文字` | 矩形範囲を指定した文字で埋める |

**例：** 複数行の先頭に `#`（スペース付き）を一括追加する

```text
Ctrl + v → 対象行を選択（j で下に） → I → # （スペース） → Esc
```

### テキストオブジェクト（強力な選択機能）

`d`, `c`, `y` などの後に組み合わせて使う。

| コマンド | 説明                                       |
| -------- | ------------------------------------------ |
| `diw`    | カーソル上の単語を削除（前後の空白を除く） |
| `daw`    | カーソル上の単語を削除（前後の空白を含む） |
| `di"`    | ダブルクォート内を削除                     |
| `da"`    | ダブルクォートごと削除                     |
| `di'`    | シングルクォート内を削除                   |
| `di(`    | 括弧 `()` 内を削除                         |
| `da(`    | 括弧 `()` ごと削除                         |
| `di[`    | 角括弧 `[]` 内を削除                       |
| `di{`    | 波括弧 `{}` 内を削除                       |
| `dit`    | HTMLタグの中身を削除                       |
| `dat`    | HTMLタグごと削除                           |
| `cit`    | HTMLタグの中身を変更                       |
| `yit`    | HTMLタグの中身をコピー                     |

**例：** `"Hello World"` の中身だけを書き換える

```text
カーソルを " の中に置いて
ci" → 新しいテキストを入力 → Esc
```

### マクロ（繰り返し操作の記録）

| コマンド | 説明                                       |
| -------- | ------------------------------------------ |
| `qa`     | レジスタ `a` にマクロの記録を開始          |
| `q`      | マクロの記録を終了                         |
| `@a`     | レジスタ `a` のマクロを実行                |
| `@@`     | 直前に実行したマクロを再実行               |
| `数字@a` | マクロを指定した回数繰り返す（例：`10@a`） |

**例：** 各行の末尾に `;` を追加するマクロ

```text
qa → A → ; → Esc → j → q
（レジスタ a に「行末に ; を追加して次の行へ」を記録）

@a → 1回実行
10@a → 10行に適用
```

**例：** 2つのファイルを並べて編集する

```bash
vim file1.md

```

"ap → レジスタ a を貼り付け

```

**例：** 離れた2か所を素早く行き来する

ma → 1か所目にマーク a を設定
mb → 2か所目にマーク b を設定
`a → 1か所目に移動
`b → 2か所目に移動
```

### コマンドラインの便利機能

| コマンド        | 説明                                        |
| --------------- | ------------------------------------------- |
| `:!コマンド`    | シェルコマンドを実行（例：`:!ls`）          |
| `:r !コマンド`  | シェルコマンドの出力を現在行の下に挿入      |
| `:r ファイル名` | ファイルの内容を現在行の下に挿入            |
| `:%!コマンド`   | ファイル全体をコマンドでフィルタリング      |
| `q:`            | コマンド履歴をウィンドウで表示・編集        |
| `Ctrl + r + "`  | インサートモードでレジスタ `"` の内容を挿入 |

**例：** 現在編集中のMarkdownファイルにシェルの `date` コマンドの出力を挿入する

```text
:r !date
```

**例：** ファイル全体を `sort` コマンドで並べ替える

```text
:%!sort
```

### 折り畳み（Fold）

| コマンド  | 説明                             |
| --------- | -------------------------------- |
| `zc`      | 現在位置の折り畳みを閉じる       |
| `zo`      | 現在位置の折り畳みを開く         |
| `za`      | 折り畳みのトグル（開閉切り替え） |
| `zR`      | すべての折り畳みを開く           |
| `zM`      | すべての折り畳みを閉じる         |
| `zf数字j` | 現在行から指定行数を折り畳む     |

---

### ヘルプ

| コマンド           | 説明                                             |
| ------------------ | ------------------------------------------------ |
| `:help`            | ヘルプを開く                                     |
| `:help コマンド名` | 指定したコマンドのヘルプを開く（例：`:help dd`） |
| `:help :set`       | `:set` コマンドのヘルプを開く                    |
| `Ctrl + ]`         | ヘルプ内のリンクにジャンプ                       |
| `Ctrl + o`         | ヘルプ内でひとつ前に戻る                         |

# Vim

Vim コマンドをまとめたリファレンスです。  
「モードの概念」を最初に把握してから、各セクションを参照してください。

---

## モードを理解する

Vim には用途ごとに切り替えるモードがあります。現在どのモードにいるかを意識することが Vim 習得の第一歩です。

| モード | 役割 | 入り方 |
| --- | --- | --- |
| ノーマルモード | 移動・削除・コピーなどの操作 | `Esc` |
| インサートモード | 文字を入力する | `i` / `a` / `o` など |
| ビジュアルモード | テキストを選択する | `v` / `V` / `Ctrl+v` |
| コマンドラインモード | 保存・検索・置換などを実行する | `:` |

!!! note "迷ったら Esc"
    どのモードにいるかわからなくなったら `Esc` を押してノーマルモードに戻りましょう。  
    `Esc` はノーマルモードで押しても何も起こらないので、何度押しても安全です。

## ファイルを開く・保存する・終了する

```text
vim ファイル名         # ファイルを開く
:w                    # 上書き保存
:w ファイル名          # 別名で保存
:q                    # 終了（変更なし）
:q!                   # 変更を破棄して強制終了
:wq                   # 保存して終了
ZZ                    # 保存して終了（ショートカット）
:e ファイル名          # 別のファイルを開く
```

!!! note "`:wq` と `ZZ` の違い"
    どちらも「保存して終了」ですが、`ZZ` はファイルに変更がない場合は保存をスキップします。  
    必ず保存したい場合は `:wq` を使いましょう。

### 使用例

```bash
# 新規ファイルを作成して開く
vim index.md

# 開いたら i でインサートモードへ → 文章を入力 → Esc → :wq で保存して終了
```

## モードを切り替える

```text
i        # カーソルの前からインサートモードへ
I        # 行頭からインサートモードへ
a        # カーソルの後ろからインサートモードへ
A        # 行末からインサートモードへ
o        # 現在行の下に新しい行を追加してインサートモードへ
O        # 現在行の上に新しい行を追加してインサートモードへ
s        # カーソル下の文字を削除してインサートモードへ
S        # 現在行を削除してインサートモードへ
Esc      # ノーマルモードに戻る
```

!!! note "i と a の使い分け"
    `i` はカーソルの**前**から、`a` はカーソルの**後ろ**から入力が始まります。  
    行末に追記したい場合は `A`、行の先頭から書き直したい場合は `I` が便利です。

### 使用例

```text
# 行末に「。」を追加する
A → 。 → Esc

# 現在行の下に新しい見出しを追加する
o → ## 新しい見出し → Esc
```

## カーソルを移動する

### 基本の移動

```text
h        # 左へ移動
j        # 下へ移動
k        # 上へ移動
l        # 右へ移動
0        # 行頭へ移動
^        # 行頭の最初の文字へ移動（空白を除く）
$        # 行末へ移動
gg       # ファイルの先頭へ移動
G        # ファイルの末尾へ移動
数字G    # 指定した行番号へ移動（例: 100G）
```

#### 使用例

```text
# 100行目へジャンプする
100G

# ファイルの末尾へ移動して追記する
G → o → テキスト入力 → Esc
```

---

### 単語・画面単位で移動する

```text
w        # 次の単語の先頭へ移動
b        # 前の単語の先頭へ移動
e        # 次の単語の末尾へ移動
H        # 画面の一番上の行へ移動
M        # 画面の中央の行へ移動
L        # 画面の一番下の行へ移動
Ctrl+f   # 1画面下へスクロール
Ctrl+b   # 1画面上へスクロール
Ctrl+d   # 半画面下へスクロール
Ctrl+u   # 半画面上へスクロール
zz       # カーソル行を画面中央に表示
zt       # カーソル行を画面上部に表示
zb       # カーソル行を画面下部に表示
```

### 行内の文字へジャンプする

```text
f文字    # 行内の次の指定文字へジャンプ（例: f, で次の , へ）
F文字    # 行内の前の指定文字へジャンプ
t文字    # 行内の次の指定文字の直前へジャンプ
T文字    # 行内の前の指定文字の直後へジャンプ
;        # f / F / t / T の検索を同方向に繰り返す
,        # f / F / t / T の検索を逆方向に繰り返す
%        # 対応する括弧 () / [] / {} へジャンプ
```

!!! note "ジャンプ履歴を使う"
    `Ctrl+o` でジャンプ前の位置に戻り、`Ctrl+i` で進めます。  
    マクロやグローバル検索など、大きく移動した後に元の位置に戻りたいときに便利です。

#### 使用例

```text
# 行内の次の , へ移動し、さらに次の , へ繰り返す
f, → ; → ;
```

## テキストを削除する

```text
x        # カーソル下の1文字を削除
X        # カーソルの前の1文字を削除
dd       # 現在行を丸ごと削除
数字dd   # 指定した行数を削除（例: 3dd で3行削除）
dw       # カーソル位置から次の単語まで削除
d$       # カーソル位置から行末まで削除（D でも同様）
d0       # カーソル位置から行頭まで削除
dG       # カーソル位置からファイル末尾まで削除
dgg      # カーソル位置からファイル先頭まで削除
```

!!! note "削除したテキストはレジスタに保存される"
    `dd` などで削除したテキストは自動的にレジスタに保存されます。  
    そのまま `p` で貼り付けることができるため、「切り取り → 貼り付け」として使えます。

### 使用例

```text
# 現在行から5行まとめて削除する
5dd

# カーソル位置から行末まで削除して書き直す（ノーマルモードのまま）
D → i → 新しいテキストを入力
```

## テキストをコピー・貼り付けする

```text
yy       # 現在行をコピー（ヤンク）
Y        # 現在行をコピー（yy と同じ）
数字yy   # 指定した行数をコピー（例: 3yy で3行コピー）
yw       # カーソル位置から次の単語までコピー
y$       # カーソル位置から行末までコピー
p        # カーソルの後ろ（下の行）に貼り付け
P        # カーソルの前（上の行）に貼り付け
```

### 使用例

```text
# 現在行を複製する
yy → p

# 3行をコピーして別の場所に貼り付ける
3yy → 移動先へ → p
```

## テキストを変更する（チェンジ）

```text
cw       # カーソル位置から次の単語まで変更（削除してインサートモードへ）
cc       # 現在行を変更
C        # カーソル位置から行末まで変更
r文字    # カーソル下の1文字を指定した文字に置き換え
R        # 上書きモードに入る（Esc で終了）
~        # カーソル下の文字の大文字・小文字を切り替え
g~~      # 現在行全体の大文字・小文字を切り替え
guu      # 現在行全体を小文字に変換
gUU      # 現在行全体を大文字に変換
J        # 現在行と次の行を結合（間にスペース）
gJ       # 現在行と次の行を結合（スペースなし）
```

!!! note "c コマンドはインサートモードに入る"
    `cw` や `cc` は削除後にインサートモードへ移行します。  
    削除だけしてインサートモードに入りたくない場合は `dw` / `dd` を使いましょう。

### 使用例

```text
# 単語を素早く書き換える
カーソルを単語の先頭に置いて → cw → 新しい単語を入力 → Esc

# 行全体を書き直す
cc → 新しいテキストを入力 → Esc
```

## 操作を元に戻す・やり直す

```text
u        # 直前の操作を元に戻す（アンドゥ）
Ctrl+r   # やり直す（リドゥ）
U        # 現在行への変更をすべて元に戻す
.        # 直前の操作を繰り返す
```

!!! note ". （ドット）コマンドは強力"
    `.` は「直前に行った操作をそのまま繰り返す」コマンドです。  
    例えば `dd` で1行削除した後に `.` を押すと、さらに1行削除されます。  
    繰り返し作業を大幅に短縮できます。

### 使用例

```text
# 誤って行を削除してしまった場合
u

# 同じ単語の置き換えを複数箇所に適用する
cw → 新しい単語 → Esc → 次の場所に移動 → .
```

## 検索・置換する

```text
/検索ワード    # 下方向に検索
?検索ワード    # 上方向に検索
n              # 次の検索結果に移動
N              # 前の検索結果に移動
*              # カーソル下の単語を下方向に検索
#              # カーソル下の単語を上方向に検索
```

```text
:s/旧/新/         # 現在行の最初のマッチを置換
:s/旧/新/g        # 現在行のすべてのマッチを置換
:%s/旧/新/g       # ファイル全体のすべてのマッチを置換
:%s/旧/新/gc      # ファイル全体を確認しながら置換
:%s/旧/新/gi      # 大文字・小文字を区別せずにファイル全体を置換
```

!!! note "確認しながら置換（/gc オプション）"
    `/gc` を末尾に付けると、1件ずつ確認しながら置換できます。  
    確認時のキー: `y`（置換）/ `n`（スキップ）/ `a`（残りをすべて置換）/ `q`（終了）

### 使用例

```text
# ファイル全体の「foo」を「bar」に置換する
:%s/foo/bar/g

# 大文字・小文字を無視して確認しながら置換する
:%s/foo/bar/gci

# カーソル下の単語を検索する
*（またはカーソルを単語に置いて #）
```

## 選択する（ビジュアルモード）

### ビジュアルモードを使う

```text
v          # 文字単位のビジュアルモード
V          # 行単位のビジュアルモード
Ctrl+v     # 矩形（ブロック）ビジュアルモード
gv         # 直前の選択範囲を再選択

# 選択後に使えるコマンド
y          # コピー
d          # 削除
c          # 変更（削除してインサートモードへ）
>          # インデント（右）
<          # インデント解除（左）
~          # 大文字・小文字を切り替え
u          # 小文字に変換
U          # 大文字に変換
```

#### 使用例

```text
# 3行をまとめてインデントする
V → jj → >

# 選択範囲をすべて大文字にする
V → 選択 → U
```

---

### テキストオブジェクトで選択する

`d` / `c` / `y` / `v` の後に組み合わせて使います。

| コマンド | 説明 |
| --- | --- |
| `diw` | カーソル上の単語を削除（空白を除く） |
| `daw` | カーソル上の単語を削除（前後の空白を含む） |
| `di"` | ダブルクォート内を削除 |
| `da"` | ダブルクォートごと削除 |
| `di'` | シングルクォート内を削除 |
| `di(` | 括弧 `()` 内を削除 |
| `da(` | 括弧 `()` ごと削除 |
| `di[` | 角括弧 `[]` 内を削除 |
| `di{` | 波括弧 `{}` 内を削除 |
| `dit` | HTML タグの中身を削除 |
| `dat` | HTML タグごと削除 |
| `cit` | HTML タグの中身を変更 |
| `yit` | HTML タグの中身をコピー |

!!! note "i と a の違い（inner / around）"
    `i` は「内側のみ」、`a` は「囲み文字ごと」を操作します。  
    例: `di"` は `"Hello"` → `""` に、`da"` は `"Hello"` ごと削除します。

#### 使用例

```text
# "Hello World" の中身だけを書き換える
カーソルを " の中に置いて → ci" → 新しいテキスト → Esc

# () の中身をコピーして別の場所に貼り付ける
カーソルを () の中に置いて → yi( → 移動先へ → p
```

---

### 矩形選択を使う

```text
Ctrl+v → 選択 → I      # 選択した各行の先頭に文字を挿入（Esc で確定）
Ctrl+v → 選択 → A      # 選択した各行の末尾に文字を挿入（Esc で確定）
Ctrl+v → 選択 → d      # 矩形範囲を削除
Ctrl+v → 選択 → r文字  # 矩形範囲を指定した文字で埋める
```

!!! note "複数行への一括操作が得意"
    矩形ビジュアルモード（`Ctrl+v`）は、複数行の同じ列位置にまとめて文字を挿入・削除するときに便利です。  
    `I` で挿入 → テキストを入力 → `Esc` で全行に反映されます。

#### 使用例

```text
# 複数行の先頭に「# 」（シャープ＋スペース）を一括追加する
Ctrl+v → 対象行を j で選択 → I → #（スペース） → Esc

# 複数行の特定の列を削除する
Ctrl+v → 列を選択 → d
```

## マクロを記録・実行する

```text
qa         # レジスタ a にマクロの記録を開始
q          # マクロの記録を終了
@a         # レジスタ a のマクロを実行
@@         # 直前に実行したマクロを再実行
数字@a     # マクロを指定した回数繰り返す（例: 10@a）
```

!!! note "マクロはどんな操作でも記録できる"
    マクロはノーマルモード・インサートモード・コマンドラインモードの操作をすべて記録します。  
    記録中は画面下部に `recording @a` と表示されます。

### 使用例

```text
# 各行の末尾に「;」を追加するマクロを記録して実行する
qa → A → ; → Esc → j → q
（「行末に ; を追加して次の行へ移動」を記録）

@a    # 1回実行
10@a  # 10行に適用
```

## レジスタを使う

```text
"ayy       # 現在行をレジスタ a にコピー
"ap        # レジスタ a の内容を貼り付け
"byy       # 現在行をレジスタ b にコピー
"bp        # レジスタ b の内容を貼り付け
"+y        # システムのクリップボードにコピー
"+p        # システムのクリップボードから貼り付け
:registers # すべてのレジスタの内容を表示
```

!!! note "複数のテキストを同時に保持できる"
    通常の `yy` はレジスタ `"` に上書きされますが、  
    名前付きレジスタ（`a`〜`z`）を使うと複数のテキストを同時に保持して貼り付けできます。

### 使用例

```text
# 2つのテキストをそれぞれ保持して貼り付ける
"ayy  # 1行目をレジスタ a にコピー
"byy  # 別の行をレジスタ b にコピー
"ap   # レジスタ a を貼り付け
"bp   # レジスタ b を貼り付け
```

## 便利な設定を変更する（:set）

```text
:set number          # 行番号を表示
:set nonumber        # 行番号を非表示
:set relativenumber  # 相対行番号を表示
:set hlsearch        # 検索結果をハイライト
:nohlsearch          # 検索ハイライトを消す（:noh でも可）
:set ignorecase      # 検索時に大文字・小文字を無視
:set incsearch       # インクリメンタルサーチを有効化
:set wrap            # 長い行を折り返して表示
:set nowrap          # 長い行を折り返さずに表示
:set tabstop=4       # タブ幅を4スペースに設定
:set expandtab       # タブをスペースに変換
:set autoindent      # 自動インデントを有効化
:set paste           # ペーストモード（自動インデントを一時無効化）
:set nopaste         # ペーストモードを解除
:set list            # 不可視文字（タブ・改行）を表示
:set nolist          # 不可視文字を非表示
```

!!! note "ペーストモードについて"
    ターミナルから長い文章を貼り付けると自動インデントが誤作動することがあります。  
    `:set paste` を先に実行してから貼り付けると、インデントが崩れません。  
    貼り付け後は `:set nopaste` で元に戻しましょう。

## vimrc を設定する

`~/.vimrc` に設定を書くと、Vim 起動時に自動的に読み込まれます。

```bash
# vimrc を開く（存在しない場合は新規作成される）
vim ~/.vimrc
```

以下はよく使う設定のサンプルです。

```vim
" ===========================
" 基本設定
" ===========================

" 行番号を表示する
set number

" 相対行番号を表示する（j/k で何行移動するか把握しやすい）
set relativenumber

" 構文ハイライトを有効化する
syntax on

" カーソル行をハイライトする
set cursorline

" ===========================
" 検索設定
" ===========================

" 検索結果をハイライトする
set hlsearch

" インクリメンタルサーチ（入力しながらリアルタイム検索）
set incsearch

" 大文字・小文字を区別しない（大文字が含まれる場合は区別する）
set ignorecase
set smartcase

" ===========================
" インデント設定
" ===========================

" タブをスペースに展開する
set expandtab

" タブ幅を4スペースにする
set tabstop=4
set shiftwidth=4

" 自動インデントを有効にする
set autoindent
set smartindent

" ===========================
" 表示設定
" ===========================

" 不可視文字を表示する（タブ・行末スペース）
set list
set listchars=tab:»-,trail:·,eol:↲

" 長い行を折り返さずに表示する
set nowrap

" ステータスラインを常に表示する
set laststatus=2

" コマンドラインの補完を有効にする
set wildmenu

" ===========================
" 操作設定
" ===========================

" バックスペースを使いやすくする
set backspace=indent,eol,start

" クリップボードと連携する
set clipboard=unnamedplus

" マウスを有効にする（ターミナルでマウス操作できる）
set mouse=a

" ファイルの変更を自動的に読み込む
set autoread

" ===========================
" キーマッピング
" ===========================

" Esc の代わりに jj でノーマルモードに戻る
inoremap jj <Esc>

" 検索ハイライトを Esc 2回押しで消す
nnoremap <Esc><Esc> :nohlsearch<CR>

" Ctrl+s で保存する
nnoremap <C-s> :w<CR>
inoremap <C-s> <Esc>:w<CR>
```

!!! note "設定を反映させるには"
    vimrc を編集した後は Vim を再起動するか、`:source ~/.vimrc` を実行すると設定が反映されます。

## プラグインを管理する（vim-plug）

### vim-plug をインストールする

=== "Linux / macOS"

    ```bash
    # vim-plug をダウンロードしてインストールする
    curl -fLo ~/.vim/autoload/plug.vim --create-dirs \
        https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim
    ```

=== "Windows（PowerShell）"

    ```powershell
    # vim-plug をダウンロードしてインストールする
    iwr -useb https://raw.githubusercontent.com/junegunn/vim-plug/master/plug.vim |`
        ni $HOME/vimfiles/autoload/plug.vim -Force
    ```

---

### プラグインを設定する

インストール後、`~/.vimrc` に以下の形式でプラグインを追記します。

```vim
" ===========================
" vim-plug プラグイン設定
" ===========================

" プラグインの読み込み開始（~/.vim/plugged にインストールされる）
call plug#begin('~/.vim/plugged')

" ファイラー：ディレクトリツリーを表示する
Plug 'preservim/nerdtree'

" ステータスバーを見やすくする
Plug 'vim-airline/vim-airline'

" Git の変更箇所を左端に表示する
Plug 'airblade/vim-gitgutter'

" ファイルをあいまい検索して開く
Plug 'ctrlpvim/ctrlp.vim'

" コメントアウトを簡単にする（gcc で行コメント）
Plug 'tpope/vim-commentary'

" 括弧・クォートの操作を拡張する（cs"' で " を ' に変換など）
Plug 'tpope/vim-surround'

" カラーテーマ
Plug 'morhetz/gruvbox'

" プラグインの読み込み終了
call plug#end()

" ===========================
" プラグインの個別設定
" ===========================

" カラーテーマを適用する
colorscheme gruvbox
set background=dark

" NERDTree を Ctrl+n で開閉する
nnoremap <C-n> :NERDTreeToggle<CR>
```

---

### プラグインをインストール・更新・削除する

```text
:PlugInstall    # vimrc に記載したプラグインをインストールする
:PlugUpdate     # インストール済みプラグインを更新する
:PlugClean      # vimrc から削除されたプラグインをアンインストールする
:PlugStatus     # プラグインの状態を表示する
:PlugUpgrade    # vim-plug 自体を更新する
```

!!! note "プラグインの追加手順"
    1. `~/.vimrc` の `call plug#begin()` と `call plug#end()` の間に `Plug 'ユーザー名/リポジトリ名'` を追記する
    2. Vim を再起動するか `:source ~/.vimrc` を実行する
    3. `:PlugInstall` を実行してインストールする

# Visual Studio Code

VSCodeを効率的に操作するための主要なコマンドとショートカットをまとめています。

## 基本コマンド

まずはこれだけは覚えておきたい、基本の操作です。

### コマンドパレットの表示

VSCodeの全機能にアクセスするための入り口です。

```text
Ctrl + Shift + P
```

**使い方例**: 「theme」と入力してエディタの配色を変えたり、各種設定を検索したりする際に使用します。

### ファイルを開く

プロジェクト内のファイルを素早く検索して開きます。

```text
Ctrl + P
```

**使い方例**: ファイル名の一部を入力するだけで、フォルダ階層を辿ることなく目的のファイルに移動できます。

### ファイルの保存

現在編集中の内容を保存します。

```text
Ctrl + S
```

**使い方例**: コードを書いた後、このコマンドでこまめに保存する癖をつけることでデータ消失を防ぎます。

### サイドバーの表示と非表示

エディタ画面を広く使いたい時に、左側のサイドバーを開閉します。

```text
Ctrl + B
```

**使い方例**: コードの記述に集中したい時はサイドバーを隠し、ファイル操作が必要な時に再度表示させます。

## よく使うコマンド

日常的なコーディング作業で頻繁に使用するコマンドです。

### ターミナルの切り替え

統合ターミナルを表示または非表示にします。

```text
Ctrl + J
```

**使い方例**: パッケージの追加やスクリプトを実行する際、マウスを使わずに素早くターミナルへアクセスします。

### 行の複製

選択中の行、またはカーソルがある行をそのまま上下にコピーします。

```text
Shift + Alt + ↓ (または ↑)
```

**使い方例**: 似たような構造のコード（リストの要素や変数定義など）を量産する際に便利です。

### マルチカーソル（複数箇所の選択）

同じ単語を一度に複数選択して編集します。

```text
Ctrl + D
```

**使い方例**: 変数名を一括で変更したい時、単語を選択した状態でこのコマンドを連打して複数選択し、一気に書き換えます。

### 定義へ移動

関数や変数が定義されている元の場所にジャンプします。

```text
F12
```

**使い方例**: 他のファイルで定義された関数の詳しい処理内容を、すぐに確認したい時に使用します。

## 上級者向けコマンド

開発効率をさらに高めるためのコマンドです。

### シンボルの名前変更

プロジェクト全体で変数名や関数名を安全に変更します。

```text
F2
```

**使い方例**: 変数名を変えたい時、手動で置換するのではなくこのコマンドを使うことで、参照されている箇所をすべて正確に書き換えられます。

### キーボードショートカット設定の表示

自分好みのショートカットを作成・変更するための画面を開きます。

```text
Ctrl + K , Ctrl + S
```

**使い方例**: よく使う拡張機能のコマンドなどを、独自の押しやすいキーに割り当てたい時に開きます。

### 設定ファイルの直接編集

GUIの設定画面ではなく、設定ファイル（JSON）をターミナルから直接編集します。

```bash
# プロジェクト固有の設定ファイルをvimで開いて編集する例
vim .vscode/settings.json
```

**使い方例**: エディタの挙動を細かくチューニングするために、設定ファイルを直接 `vim` で開いて編集・保存します。

## 知ってると便利なコマンド

作業のストレスを減らしてくれる便利な機能です。

### エディタの分割

画面を左右（または上下）に分割して、複数のファイルを同時に表示します。

```text
Ctrl + \
```

**使い方例**: 左側でテストコードを書き、右側で実装コードを書くなど、見比べながら作業する際に役立ちます。

### Markdownプレビューの表示

編集中のMarkdownファイルがどのように表示（レンダリング）されるかを確認します。

```text
Ctrl + Shift + V
```

**使い方例**: MkDocsに反映させる前に、見出しの階層やコードブロックが正しく表示されているかチェックします。

### 行の移動

選択した行を丸ごと上下に移動させます。

```text
Alt + ↑ (または ↓)
```

**使い方例**: コードの順番を入れ替えたい時、カット＆ペーストの手間をかけずに直感的に配置を変えられます。

### プロジェクト内の全検索

フォルダ内のすべてのファイルから特定の文字列を探し出します。

```text
Ctrl + Shift + F
```

**使い方例**: 特定の関数がプロジェクトのどこで使われているか、全体を網羅的に調べたい時に使用します。
